Summary: Load stock ticker data for analysis, with options to specify the date range, interval, data source, and file format. You can also choose to include pre/after market hours, load monthly or weekly data, and display performance information. Various export options and data sources are available.

Examples:
- Load stock data for <SYMBOL> with default settings: stocks/load -t <SYMBOL>
- Load stock data for <SYMBOL> with a custom date range: stocks/load -t <SYMBOL> -s 2022-01-01 -e 2022-12-31
- Load stock data for <SYMBOL> with 15-minute intervals: stocks/load -t <SYMBOL> -i 15
- Load stock data for <SYMBOL> including pre/after market hours: stocks/load -t <SYMBOL> -p
- Load stock data for <SYMBOL> from a custom file: stocks/load -t <SYMBOL> -f FILEPATH
- Load monthly data for <SYMBOL>: stocks/load -t <SYMBOL> -m
- Load weekly data for <SYMBOL>: stocks/load -t <SYMBOL> -w
- Load stock data for <SYMBOL> and show performance information: stocks/load -t <SYMBOL> --performance
- Load stock data for <SYMBOL> and export in json format: stocks/load -t <SYMBOL> --export json
- Load stock data for <SYMBOL> using a specific data source: stocks/load -t <SYMBOL> --source AlphaVantage
