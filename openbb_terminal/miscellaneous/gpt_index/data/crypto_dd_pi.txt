Display asset's project info [Source: https://messari.io]

pi
Examples:
- To display the project information of a cryptocurrency: crypto/load <COIN>/dd/pi

- Access the project details for a particular crypto asset: crypto/load <COIN>/dd/pi
- Obtain the project information of a specific crypto coin: crypto/load <COIN>/dd/pi
