Summary: Retrieve dark pool short positions for stocks with options to limit the number of tickers displayed, sort by specific fields, and export the data in various formats. [Source: Stockgrid]

Examples:
- Display the top 10 dark pool short positions sorted by default: stocks/dps/pos
- Display the top 5 dark pool short positions: stocks/dps/pos -l 5
- Display dark pool short positions sorted by Short Vol. %: stocks/dps/pos -s sv_pct
- Display dark pool short positions sorted by Net Short Vol. [1M] in ascending order: stocks/dps/pos -s nsv -r
- Export dark pool short positions data into a csv file: stocks/dps/pos --export csv
- Export dark pool short positions data into an xlsx file with a specific sheet name: stocks/dps/pos --export xlsx --sheet-name Sheet1
